It is anticipated that Portable Web Publications (PWP) for the Open Web Platform will need to be archived. [pwp-archival-services] This document describes archival use cases and requirements of potential interest to those developing the vision and requirements for PWPs.

This document is a work in progress. It is an internal document of the W3C Digital Publishing Interest Group. It has no official standing of any kind and does not represent consensus of the W3C Membership.

Introduction

Captured here are use cases and parts of use cases that provide context and articulate the requirements and expectations typical of archival services that are likely to archive PWPs. To add a use case, please copy and populate the template and insert as next new section below. The use cases are for now unordered. The template provided is adapted from the IG's Use Case Template.
